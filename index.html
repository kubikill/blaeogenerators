    Changelog:

    v2.03 - August 28, 2021

    • Bar: Mouse cursor now changes to a pointer when hovering over a bar that has review trigger method set to "bar"
    • Box: Completion bar can now be hidden
    • Box: Added minimal mode, which hides all text under the game image
    • Hero: The logo now shrinks instead of stretching if it doesn't fit on the image
    • Fixed save/load buttons being in wrong positions
    • Fixed game images not updating when inputting an unowned AppID
    • Fixed "Copied!" tooltip color

    v2.02 - September 22, 2020

    • Bar: Slightly increased top padding in reviews when using button review trigger
    • Hero: Added linebreak after hero
    • Hero: Parts of logo that are positioned outside of the banner are no longer visible
    • Fixed image spoilers not displaying properly in previews

    v2.01 - June 28, 2020

    • Added Twitch platform icon
    • Added link to BLAEO Post Saver in Help/Resources
    • Minor tweaks to platform icon sizes and positions
    • Fixed "undefined" achievements appearing during game search in some rare cases

    v2.0 - March 26, 2020

    • Added "Hero" generator
    • Games can now be autofilled by typing their name
    • Improved save/load - all generator settings are now saved, saves can now be loaded automatically on startup
    • Added support for spoiler tags in reviews
    • Added Help/Resources tab
    • Bar/Panel/Box: Added platform icons
    • Bar/Panel/Box: Game images can now link to a custom page
    • Bar/Panel: Added "always visible" reviews
    • Bar/Panel: Added ability to use 3 colors for gradient backgrounds
    • Bar: "Click to expand" is now displayed on the bar if the "bar" review trigger method is used
    • Bar: Game images can now be changed to: capsule (default), header, custom, none
    • Bar: Review button background and font colors can now be customized
    • Bar: Text shadows can now be disabled
    • Bar: Added option to hide playtime and achievements
    • Bar: Bar height can now be changed
    • Panel: Added a custom text field
    • Panel: The rating now won't display if the rating field is empty
    • Panel: Background color and text color can now be customized
    • Panel: Game images can now be changed to header or custom
    • Panel: The completion bar can now be hidden
    • Panel: Icon color can now be changed
    • Box: Added screenshot icon
    • Box: Background and text colors can now be changed
    • Box: Added background presets
    • Box: Game images can now be changed to: capsule (default), header, custom
    • Progress bar: Added option to change rounding of the percents
    • Progress bar: You can now apply decoration to all parts of the bar at once
    • Improved interface
    • Major code rewrite

    v1.21 - March 29, 2019

    • Bar and Panel: Added screenshot button
    • Added "Copy with linebreaks to clipboard" buttons to export easier to read code
    • Added hex color codes to "Completion bar color"

    v1.2 - May 25, 2018

    • Added a panel generator
    • Added Markdown support to reviews
    • Added automatic and manual saving for reviews
    • Bar: Added "click button to view review"
    • Progress bar: Added striped/animated bars
    • Miscellaneous small changes

    v1.1 - April 21, 2018

    • Added a progress bar generator
    • Bar: Added reviews
    • Bar: Added option to change game image position
    • Bar: Added option to change completion bar position
    • Bar: Text color can now be set separately for title and playtime/achievements
    • Bar: Text color option now affects achievement link
    • Changed how games with no achievements are handled
    • Fixed achievement text showing "undefined of x achievements", if user has 0 achievements in a game
    • Fixed fetching games with progress set to "never played" or "won't play"
    • Implemented some tiny changes I don't remember anymore
    • Removed the "preview always visible" option because it breaks review previewing ¯\_(ツ)_/¯

    v1.01 - March 27, 2018

    • Added an one-time welcoming message
    • Added "never played (white)" to completion bar colors
